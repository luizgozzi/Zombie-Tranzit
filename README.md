# Zombie Tranzit
Um endless runner de fugir de zumbis feito em Unity.

## Versão
- Unity 6.3 LTS
- Windows Build Support (IL2CPP)

## Como rodar
1. Clone este repositório
2. Abra no Unity Hub
3. Use a versão indicada
4. Rode a cena principal

## Objetivo
Desviar dos obstáculos e sobreviver o máximo possível.
